# Hana Nováčková - About me

#### Description
A clean, responsive personal website built with semantic HTML5 and modern CSS. Features a three-column layout optimized for both desktop and mobile viewing.

#### Design
- Dark Theme: Elegant dark gray color scheme with high-contrast typography
- Card-Based Layout: Each section uses rounded corners and subtle shadows for visual hierarchy
- Responsive Design: Mobile-first approach with breakpoints at 768px
- CSS Variables: Centralized color management through custom properties

#### Project Structure

```
├── index.html          # Main HTML document with semantic markup
├── default.css         # Comprehensive stylesheet with responsive design
├── README.en.md        # English documentation
├── README.md           # Chinese documentation
├── images/             # All visual assets (background, avatars, icons)
├── Documents/          # Supplementary documentation
├── LICENSE             # Open source license
└── .css                # Project styling configuration
```

#### Technical Stack
- HTML5: Semantic markup for accessibility and SEO
- CSS3: Modern features including Grid, Flexbox, and CSS Variables
- Responsive Design: Mobile-first approach with media queries
- Cross-Browser: Compatible with modern web browsers

#### Quick Start

1.  Clone the repository
2.  Open index.html in any modern browser
3.  No build process or dependencies required
